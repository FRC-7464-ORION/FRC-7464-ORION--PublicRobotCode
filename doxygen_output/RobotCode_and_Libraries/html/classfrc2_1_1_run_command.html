Detailed Description

A command that runs a Runnable continuously. Has no end condition as-is; either subclass it or use Command.WithTimeout() or Command.WithInterrupt() to give it one. If you only wish to execute a Runnable once, use InstantCommand.

This class is provided by the NewCommands VendorDep

void wpi::uv::Poll::Reuse ( int  fd,
std::function< void()>  callback 
)

Reuse this handle. This closes the handle, and after the close completes, reinitializes it (identically to Create) and calls the provided callback. Unlike Close(), it does NOT emit the closed signal, however, IsClosing() will return true until the callback is called. This does nothing if IsClosing() is true (e.g. if Close() was called).

FRC Team 7464 - O.R.I.O.N.: Code for Sirius (2020 Season)

Introduction

This is the documentation for the software code written for Sirius, the Oklahoma Robotic Institute of Norman's (O.R.I.O.N.'s) entry for competition in the FIRST Robotic Competition (FRC) 2020 season.

This documentation was created by adding special comments to our computer code for Sirius, which is written in C++. A program called Doxygen is used to examine the source code for these special comments, and generates this documentation.

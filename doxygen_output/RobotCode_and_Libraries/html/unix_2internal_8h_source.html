179  * when O_NDELAY is not equal to O_NONBLOCK. Case in point: linux/sparc32
180  * and linux/sparc64, where O_NDELAY is O_NONBLOCK + another bit.
181  *
182  * Libuv uses uv__nonblock_fcntl() directly sometimes so ensure that it
183  * commutes with uv__nonblock().

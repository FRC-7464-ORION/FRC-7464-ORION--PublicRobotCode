◆ CreateRpcCallPoller()

NT_RpcCallPoller nt::CreateRpcCallPoller ( NT_Inst  inst)

Create a RPC call poller. Only valid to use on the server.

A poller provides a single queue of poll events. Events linked to this poller (using CreatePolledRpc()) will be stored in the queue and must be collected by calling PollRpc(). The returned handle must be destroyed with DestroyRpcCallPoller().

Parameters
instinstance handle
Returns
poller handle

◆ PollRpc() [1/2]

std::vector< RpcAnswer > nt::PollRpc ( NT_RpcCallPoller  poller)

Get the next incoming RPC call. This blocks until the next incoming RPC call is received. This is intended to be used with CreatePolledRpc(); RPC calls created using CreateRpc() will not be serviced through this function. Upon successful return, PostRpcResponse() must be called to send the return value to the caller.

Parameters
pollerpoller handle
Returns
Information on the next RPC calls. Only returns empty if an error occurred (e.g. the instance was invalid or is shutting down).

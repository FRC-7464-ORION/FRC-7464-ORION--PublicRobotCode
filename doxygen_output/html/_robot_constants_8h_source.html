150 // The joystick/motor curve parameters should make it easier to control
151 // motors with a joystick and/or thumbstick and/or other variable
152 // control (such as the triggers on some controllers).
153 //
154 // Here is what each parameter means/does:
155 //
156 // Null Zone: Sometimes called deadzone. Any absolute value output from the
157 // joystick/thumbstick/trigger/etc. that is less than the
158 // null zone value will translate into zero output onto a motor.
159 // This keeps the motor from "jittering", especially if the
160 // output of the controller is noisy (or a nervous driver ;-).
161 //
162 // Typical values for a null zone are from 0.0 to 0.2.
163 // The null zone cannot exceed 1.0 (nothing will work!).
164 //
165 // Max Speed: This is the maximum speed that the motor will output
166 // (either forward or reverse). Any absolute value output from the
167 // joystick/thumbstick/trigger/etc. that is greater than the
168 // maximum speed will be limited to the maximum speed
169 // (either forward or reverse).
170 //
171 // Set the maximum speed to the percentage of full speed desired.
172 // For example, to limit the motor to half speed, set max speed
173 // to 50% (0.50). Similar to the null zone, don't see the
174 // max speed to 0.0, or again nothing will work!
175 //
176 // Exponent: This is the exponent of a curve used on the joystick input
177 // to motor output mapping. Its intent is to make the motor more
178 // responsive at lower speeds, at the loss of fine control
179 // at higher speeds. An exponent of 1.0 makes the output
180 // linear (like a straight line). Higher exponents provide
181 // for better control at lower speeds, but sacrifice control
182 // at higher speeds.
183 //
184 // You'll typically want the exponent to be between 2.0-4.0.
185 // Anything past 5.0 will result in an uncontrollable robot!
186 // Do not use an exponent less than 1.0. The math breaks
187 // down, and we may open a rift to another universe.
188 //
189 // See the SW_Design repository, and look at NewThumbsticksOutputs.xlsx
190 // for some graphs.
191
